1.2 Análisis de impacto

We have supported some countries with this. We don’t have good documentation yet but this is something I plan to work on soon. In the interim we can advise. Very briefly when introducing the 19th ICLS it is possible to build the questionnaire in a way which enables estimates that are theoretically comparable to the 13th ICLS definitions to be generated. However we would not suggest fully relying on this as the other changes in questionnaire content could have impacts. The critical thing is to consider the testing the implementation approach. If a parallel survey can be done this can generate very valuable information enabling deep analysis of the source of differences for communication purposes as well as potentially enabling linking of key series. I have supported some countries in this process and cold elaborate on this when needed.

For example in Georgia they introduced some questions in the ‘old’ LFS about one year before the transition. This was to enable some estimation of the kindly scale of impact which turned out to be pretty accurate for employment but not unemployment to the same degree. Then they had one quarter of a parallel survey. Using some assumptions they were able to model backwards the new series for 10 years (and could in theory carry forward the old if really desired but we would not consider ideal).

1.2.3.2 El enfoque de Di Natale

Para aplicar este ajuste, solo se necesita una serie, es decir, la serie nueva \(x.n\) Pasos:

  1. Identificar el punto donde ocurrió el cambio, el cual se denota por \(t_0\)

  2. Calcular el factor de ajuste que representa el cambio (porcentual): \[ f=\frac{x.n_{t_0}}{x.n_{t_0-1}} -1 \]

factor <- x.n[t0]/x.n[t0-1] - 1
  1. A todos los valores de la serie regular antes de \(t_0\) se multiplica por el factor proporcionalmente, así el valor \(t\) (con \(t=1,\cdots,t_0-1\)) de la serie ajustada queda definido como \[ x^{adj}_t=x.n_t* \left ( 1+f*\frac{t}{t_0-1} \right ) \]
x.nadale <- x.n
x.nadale[1:(t0-1)] <- x.n[1:(t0-1)] * (1 + factor*(1:(t0-1))/(t0-1))

Miramos la gráfica:

x$x.nadale <- x.nadale
ggplot(x) + 
  geom_line(aes(x=t, y=x.n), col = "blue") +
  geom_line(aes(x=t, y=x.nadale), col = "green") + 
  geom_text(aes(x = 130, y =0.3), col="blue", label = "serie nueva")  + 
  geom_text(aes(x = 50, y =0.5), col="green", label = "serie ajustada") + 
  ggtitle("No modifica la serie después de t0")
